🎯 教學目標

本教材以「放大鏡實驗」為核心，透過互動式 Canvas 動畫，讓國小高年級學生能夠：

  • 理解凸透鏡成像的五種情況（倒立縮小、倒立等大、倒立放大、焦點模糊、正立放大虛像）
  • 觀察三條標準光線（平行光、過焦點、過光心）的幾何路徑
  • 透過放大鏡模擬實際操作體驗，連結生活情境
  • 透過素養測驗驗收學習成果

⚙️ 基本參數設定

焦距 f20 公分（模擬台固定值）
物距範圍4 ~ 90 公分（滑桿可調）
預設物距47 公分（範圍中間值）
目標對象國小高年級（自然科學）

七大功能模組

🕯️

TAB 1 ── 蠟燭成像模擬台

Canvas 2D即時互動

使用者拖曳滑桿改變物距（4～90cm），畫面即時重繪凸透鏡、物體（蠟燭）與成像結果，並以文字顯示成像種類說明。

  • 套用薄透鏡公式計算像距 v = f·u / (u−f)
  • 依物距自動判斷五種成像狀態並顯示彩色標籤
  • 凸透鏡以雙凸 bezier 曲線繪製，F 與 2F 標記以圓點標示
  • 物距滑桿每 10cm 標示刻度，使用 <datalist> 提供吸附點
  • 附「成像規律對照表」（五列完整規律）
📊 成像判斷邏輯：u > 2f+1 → 倒立縮小 ｜ u ≈ 2f → 倒立等大 ｜ f < u < 2f → 倒立放大 ｜ u ≈ f → 模糊 ｜ u < f → 正立放大虛像
📐

TAB 2 ── 光線折射路徑動畫

Canvas 2D幾何光學

可視化三條標準光線的折射路徑，幫助學生理解成像的幾何原理。

  • 🔴 紅色：平行主軸入射 → 折射後通過右焦點 F'
  • 🟢 綠色：通過左焦點 F → 折射後平行主軸
  • 🔵 藍色：通過光心 O → 方向不變直線穿透
  • 虛像情況下，光線延伸以虛線表示（真實光線 + dashed 回溯線）
  • 後台以斜率計算延伸光線交點，精確定位成像位置

物理公式與計算

1/v + 1/u = 1/f
薄透鏡公式（Thin Lens Equation）
u = 物距（物體到透鏡距離）　v = 像距（像到透鏡距離）　f = 焦距（本教材固定為 20 cm）

程式中的計算實作

// 焦距常數
const F = 20;  // 公分

// 計算像距（薄透鏡公式變形）
const v = F * u / (u - F);
// 正值 → 實像（透鏡右側）
// 負值 → 虛像（透鏡左側）

// 像的高度（放大率）
const ih = ch * (v / u);
// |ih| > ch → 放大；|ih| < ch → 縮小
// ih < 0（v > 0，u > 0）→ 倒立

// Canvas 座標轉換（比例尺）
const sc = Math.min((W * 0.42) / F, 5);  // px/cm
const ix = lx + v * sc;  // 像的 x 座標

五種成像狀態判斷邏輯

條件成像種類說明
u > 2f+1倒立縮小實像相機、人眼原理
u ≈ 2f（±1）倒立等大實像等倍點，像與物等大
f+1 < u < 2f-1倒立放大實像投影機、幻燈機原理
u ≈ f（±2）模糊（焦點）光線平行，無法成像，可聚光點火
u < f正立放大虛像放大鏡、老花眼鏡原理

Canvas 2D 繪圖實作

凸透鏡繪製（雙凸外形）

使用二次 Bézier 曲線 quadraticCurveTo() 繪製雙凸透鏡輪廓，中間加上半透明填充，兩端保留物理圖慣例的箭頭標記。

function drawLens(c, lx, ay, lh) {
  lh = lh || 140;
  const hw = Math.min(22, lh * 0.15);  // 透鏡中央最大寬度

  // ① 雙凸輪廓（兩段 bezier 弧）
  c.beginPath();
  c.moveTo(lx, ay - lh/2);                          // 頂點
  c.quadraticCurveTo(lx+hw, ay, lx, ay + lh/2);  // 右弧
  c.quadraticCurveTo(lx-hw, ay, lx, ay - lh/2);  // 左弧
  c.closePath();
  c.fillStyle = 'rgba(21,101,192,0.15)';
  c.fill();
  c.strokeStyle = '#1565c0'; c.lineWidth = 2.5;
  c.stroke();

  // ② 頂端 / 底端箭頭（物理符號慣例）
  [[lx, ay-lh/2, lx-8, ay-lh/2+14, lx+8, ay-lh/2+14],
   [lx, ay+lh/2, lx-8, ay+lh/2-14, lx+8, ay+lh/2-14]
  ].forEach(pts => {
    c.beginPath();
    c.moveTo(pts[0], pts[1]);
    c.lineTo(pts[2], pts[3]);
    c.lineTo(pts[4], pts[5]);
    c.closePath(); c.fill();
  });
}

放大鏡效果實作（Canvas 裁切放大）

將底層場景先渲染到離屏 Canvas（offscreen canvas），再以 drawImage() 的來源裁切區域縮放到放大鏡圓形範圍內，配合 clip() 遮罩實現圓形放大效果。

// 1. 計算放大鏡圓形 clip 區域
c.save();
c.beginPath();
c.arc(mx, my, R, 0, Math.PI * 2);
c.clip();

// 2. 從離屏畫布裁切對應區域並放大繪製
const srcX = mx - R / power;
const srcY = my - R / power;
const srcW = R * 2 / power;
c.drawImage(offscreenCanvas,
  srcX, srcY, srcW, srcW,  // 來源（小範圍）
  mx-R, my-R, R*2, R*2   // 目的地（放大顯示）
);
c.restore();

// 3. 繪製放大鏡邊框與反光效果
c.strokeStyle = '#90a4ae'; c.lineWidth = 8;
c.beginPath(); c.arc(mx, my, R, 0, Math.PI*2); c.stroke();

刻度標尺（Tick Ruler）

使用 <datalist> 提供滑桿吸附點，JavaScript 動態生成數字標籤並以百分比定位。

// HTML: datalist 提供吸附點
<input type="range" id="slSim" list="physTicks">
<datalist id="physTicks">
  <option value="10"></option>
  <option value="20"></option>  /* …到 90 */
</datalist>

// JS: 動態插入標籤
function buildRuler(id, min, max) {
  const el = document.getElementById(id);
  for (let v = 10; v <= max; v += 10) {
    const pct = (v - min) / (max - min) * 100;
    const s = document.createElement('span');
    s.textContent = v;
    s.style.left = pct + '%';  // 絕對定位
    el.appendChild(s);
  }
}

響應式設計（RWD）

採用 CSS 媒體查詢 + Flexbox/Grid 自動換行，所有 Canvas 尺寸由 JS 動態計算，確保在不同裝置下正確顯示。

斷點目標裝置調整項目
> 768px桌機 / 筆電完整排版，側邊欄 + 主面板並排
≤ 768px平板字體縮小、卡片 padding 縮減、表格字縮小
≤ 480px手機單欄排版、tab-bar 橫向滾動、Canvas 全寬
📱 觸控支援：放大鏡模擬支援 touchmove / touchstart 事件，手機可用手指拖曳；模式2支援雙指縮放（pinch-to-zoom）改變放大倍率。

Canvas 自適應寬度

// 視窗調整時自動重新計算 Canvas 大小
window.addEventListener('resize', () => {
  resizeSim(); updateSim();
  resizeRay(); updateRay();
  resizeMag();
});

function resizeSim() {
  const w = simCanvas.parentElement.clientWidth - 48;
  simCanvas.width  = Math.max(w, 320);  // 最小寬度 320px
  simCanvas.height = 300;
}
