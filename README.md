## 簡介

本網站是配合「放大鏡實驗（凸透鏡成像原理）」教學影片所製作的互動數位教材，適合國小高年級自然領域教師課堂使用或學生自主學習。全站為單一 HTML 檔案，無需伺服器，可離線使用。

## 技術規格

- **單一檔案**：`index.html`（HTML + CSS + JavaScript，無外部依賴）
- **離線可用**：不需網路、不需後端
- **響應式設計（RWD）**：支援桌機、平板、手機（斷點 640px / 420px）
- **Canvas 繪圖**：成像模擬、光線路徑圖、生態海報、放大鏡效果均使用 HTML5 Canvas 2D API
- **放大鏡技術**：`clip()` + `translate()` + `scale()` 實現圓形放大效果
- **觸控支援**：touchmove 單指移動放大鏡、雙指 pinch-to-zoom 調整倍率

---

## 放大鏡模擬兩種操作模式

| 模式 | 桌機操作 | 行動裝置操作 |
|------|---------|------------|
| 🔍 放大鏡模式 | 滑鼠移動放大鏡位置，滑桿調整倍率（2×～15×） | 單指拖曳移動放大鏡 |
| 🖱️ 滾輪縮放模式 | 滑鼠移動放大鏡位置，滾輪縮放鏡內倍率（2×～15×） | 單指移動，雙指 pinch 縮放 |

## 部署方式

```bash
git clone https://github.com/prayer168/magnifier.git
cd magnifier
# 直接用瀏覽器開啟 index.html，或 push 至 GitHub Pages
```

GitHub Pages 設定：Settings → Pages → Source: Deploy from a branch → main → / (root) → Save
