# 放大鏡實驗互動學習網站（精簡版）

**科學探究與實作 ｜ 凸透鏡成像原理 ｜ 國小高年級自然領域**

線上瀏覽：<https://prayer168.github.io/magnifier-simple/>

---

## 簡介

本 repo 是 `magnifier` 教材網站的精簡版，保留可直接操作並從實驗中得到答案的核心學習活動。全站為單一 HTML 檔案，無需後端，可用 GitHub Pages 部署。

---

## 保留功能

| 頁籤 | 功能說明 |
|------|---------|
| 成像模擬 | 拖曳物距滑桿，即時觀察凸透鏡成像與成像規律對照表 |
| 光線路徑 | 三條標準光線折射動畫，補上 4-33 cm 區間的成像/無成像提示 |
| 放大鏡模擬 | 自然生態海報與互動放大鏡操作 |
| 素養測驗 | 6 題可透過本網站實驗操作推得答案的題目 |

已先移除首頁入口：

- 光學常識
- 光學小故事
- 專有名詞

---

## 本版調整

- 新 repo：`prayer168/magnifier-simple`
- 光線路徑在物距 `4-33 cm` 時仍會顯示成像狀態與文字說明。
- 嚴格依薄透鏡公式判斷成像；只有 `物距 = 焦距 = 20 cm` 時才顯示無有限成像。
- 物體高度固定，不隨滑桿或畫面縮放改變。
- 像跑到畫布外時，不壓縮比例，改以邊界提示框標出像在左方或右方較遠處，並顯示像距、像高。
- 素養測驗改為 6 題，題目答案皆可透過成像模擬、光線路徑或放大鏡模擬得到。

後續維護請先閱讀 `memory.md` 的 `2026-07-08 成像與光線路徑修正紀錄`，避免把固定物高、焦點判斷、超界提示與光線路徑規則改回舊版。

## 技術規格

- 單一檔案：`index.html`

- Canvas 2D 繪製成像模擬、光線路徑與放大鏡效果

## 部署方式

```bash
git clone https://github.com/prayer168/magnifier-simple.git
cd magnifier-simple
```

GitHub Pages 設定：

```text
Settings → Pages → Deploy from a branch → main → / (root)
```
